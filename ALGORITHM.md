# Smart Money Screener — 알고리즘 명세 v1.0

## 핵심 철학

> **"스마트머니가 매집을 끝내고 출발한 종목을 잡는다"**
>
> - 시간 스케일만 다를 뿐, 본질은 동일
> - **매집 → 출발 → 상승**의 패턴을 두 시간축으로 탐지
> - 마지막 단계에서 **단타 / 텐버거**로 분기

## 두 전략 정의

| 구분 | 매집 기간 | 결판 | 보유 |
|------|-----------|------|------|
| 💎 **텐버거** | **1~2년** (250~500 거래일) | 수개월~수년 | 길게 |
| ⚡ **단타** | 며칠~수 주 | **7일 내 폭발** | 1~7일 |
| ⭐ **황금자리** | 두 신호 동시 점등 | — | 진입 짧게·보유 길게 |

---

## 5단계 파이프라인

```
[1] 데이터 수집
       ↓
[2] 종목 유니버스 필터
       ↓
[3] 공통 분석 (수급·차트·매집)
       ↓
[4] 출발 신호 확인 (게이트 - 통과 못하면 제외)
       ↓
[5] 분류 (단타 / 텐버거 / 둘다)
```

---

## [1단계] 데이터 수집

**일별 데이터 (장 마감 후)**
- OHLCV (시가/고가/저가/종가/거래량)
- 외국인 순매수 (금액·수량)
- 기관 순매수 (세분화: 금융투자/투신/연기금/사모/보험/은행)
- 개인 순매수
- 프로그램 매매 (차익/비차익)
- 시가총액
- 공매도 잔고
- 대차잔고

**저장 구조**
```python
종목별_시계열 = {
    "ticker": "005930",
    "daily": [
        {"date": "...", "ohlcv": {...}, "수급": {...}}
    ]
}
```

## [2단계] 종목 유니버스 필터

**제외 조건**
- 거래정지·관리·투자유의 종목
- 시총 300억 미만 (유동성 부족)
- 일평균 거래대금 5억 미만
- 상장 1년 미만 (텐버거 매집 기간 검증 불가)
- 적자 + 부채비율 과다 (펀더멘털 워닝)

→ 약 1,000~1,500개 종목으로 압축

## [3단계] 공통 분석 (핵심 엔진)

### 3-1. 수급 점수 (Supply/Demand Score) — 0~100점

**A. 수급 일치도 (Confluence) — 30점**
- 외국인 + 기관 동시 순매수: +20점
- 개인 동시 순매도: +10점 (역지표)
- 프로그램 차익 제외 후 실제 순매수만 카운트

**B. 수급 강도 (Intensity) — 25점**
- 시총 대비 순매수 비율 (%): 정규화 후 환산
- 거래대금 대비 순매수 비중: 정규화 후 환산
- 단순 절대금액 X, 비율로 평가

**C. 수급 지속성 (Persistence) — 25점**
- 외국인 N일 연속 순매수
- 기관 N일 연속 순매수
- 매도 → 매수 전환 시점일 경우 +보너스 (변곡점)

**D. 보조 신호 (Aux) — 20점**
- 공매도 잔고 감소: +10점
- 대차잔고 감소: +5점
- 신규 기관 유입 (이전 분기 미보유): +5점 (O'Neil 'I')

### 3-2. 매집 단계 분석 (Wyckoff)

```python
def 매집_단계_판별(종목):
    가격_변동성 = std(종가, 기간)
    거래량_평균변화 = avg(거래량, 최근) / avg(거래량, 이전)
    가격_레인지 = (max - min) / 평균가

    if 가격_레인지 < 임계값 and 수급_점수 > 60:
        if 거래량_평균변화 < 1: return "Stage 1 (매집)"
        if 거래량_평균변화 > 1.5: return "Stage 1→2 전환 (출발)"

    return "기타"
```

**매집 기간 측정**
- 박스권 형성 시작일 ~ 현재일
- 박스권 정의: ATR 기반 ±X% 횡보 구간

### 3-3. 차트 패턴 분석

**VCP (Volatility Contraction Pattern) — Minervini**
- 박스권 폭이 점점 좁아지는 패턴
- 거래량도 함께 감소
- 압축 후 폭발

**박스 이론 (Darvas Box)**
- 명확한 저항선·지지선 형성
- 신고가 갱신마다 새 박스 형성

**Stage Analysis (Weinstein)**
- Stage 1: 바닥 횡보 (매집)
- Stage 2: 상승 추세 (마크업)
- Stage 3: 천장 횡보 (분산)
- Stage 4: 하락 추세

## [4단계] 출발 신호 확인 (필수 게이트)

**이걸 통과 못하면 모두 제외.** 매집만 보고 잡으면 너무 일찍 들어가서 시간 낭비.

### 4-1. 단타용 출발 신호 (7일 내 폭발 가능성)

```python
단타_출발 = (
    당일_거래대금 >= 평균_거래대금 * 3.0           # 거래량 폭발
    and 외국인_or_기관 순매수 전환                  # 수급 진입
    and 종가 >= (시가 + 고가) / 2                  # 강한 마감
    and 박스권_상단_돌파                            # 기술적 돌파
    and rvol >= 2.0                                # 상대 거래량
)
```

### 4-2. 텐버거용 출발 신호 (Stage 1→2 전환)

```python
텐버거_출발 = (
    매집_기간 >= 250일                             # 1년 이상 매집
    and 박스권_상단_돌파 (1~2년 박스)              # 큰 돌파
    and 거래량 >= 평균_거래량 * 2.0                # 거래량 동반
    and 60일선 > 240일선                           # 장기 정배열 전환
    and 52주_신고가_갱신                           # 신고가
    and 외국인_누적순매수_긍정 (최근 60일)         # 수급 가속
)
```

## [5단계] 분류

```python
def 분류(종목):
    라벨 = []

    if 단타_출발(종목):
        라벨.append("⚡단타")

    if 텐버거_출발(종목):
        라벨.append("💎텐버거")

    if "⚡단타" in 라벨 and "💎텐버거" in 라벨:
        라벨.append("⭐황금자리")

    return 라벨
```

## 백테스트 검증

**단타 검증**
- 출발일 다음날 시가 매수 → 5일 후 종가 매도
- 승률, 평균 수익률, 최대 손실
- 기준선: 무작위 진입 대비

**텐버거 검증**
- 출발일 종가 매수 → 6개월/1년/2년 후 종가
- 50% 이상 상승 종목 비율 (텐버거 후보)
- 최대 수익률, 최대 낙폭

**기준점 통과 조건**
- 단타 승률 55%+, 평균 수익률 양수
- 텐버거: 출발 신호 종목 중 50%+ 상승 비율 30%+

→ 통과 못하면 이론·사례 연구로 회귀
