## VCP — Volatility Contraction Pattern (핵심)

> **"폭발 직전엔 거래량이 마르고 변동폭이 좁아진다."**

### 정의
- 박스권 형성 → 박스 폭이 점점 좁아짐 → 거래량도 감소
- 보통 2~6번의 수축 발생
- 마지막 수축에서 거래량이 가장 마름

### 시각적 패턴
```
      |↑↓| 
       |↑↓|     ← 1차 수축 (예: ±25%)
        |↑↓|    ← 2차 수축 (예: ±15%)
         |↑↓|   ← 3차 수축 (예: ±8%)
          |     ← 4차 수축 (예: ±4%) ← 거의 안 움직임
          |
       BREAKOUT (거래량 폭발)
```

### 정량 조건
- 각 수축의 폭이 이전의 **40~50%** 수준으로 감소
- 거래량이 마지막 수축에서 평균 대비 **50% 이하**
- 박스권 폭이 평균 ATR의 1.0배 이내

## 진입 (Pocket Pivot & Pivot Buy)

**Pocket Pivot**: VCP 도중 박스 안에서 거래량 폭발하며 상승하는 봉
- 진입 가능 시점

**Pivot Buy**: VCP 완성 후 박스 상단 돌파
- 거래량 평균 대비 1.5~2배+
- 종가가 고가 부근

→ 우리 단타 출발 신호와 매우 유사

## 손절 (Risk Management)

> **"진입가 대비 -8% 이내 손절. 예외 없음."**
- 평균 손절 -3~4% 유지
- 익절은 천천히 (Stage 1 진행 중엔 보유)

## VCP 탐지 알고리즘 (의사코드)

```python
def detect_vcp(종가_시계열, 거래량_시계열, lookback=60):
    """VCP 패턴 탐지: 박스 폭이 점진 감소하는지"""
    # 최근 lookback 일을 4구간으로 분할
    구간 = split(종가_시계열[-lookback:], 4)
    
    # 각 구간의 변동폭 (high-low / mean) 계산
    폭들 = [(max(c) - min(c)) / mean(c) for c in 구간]
    
    # 점진 감소 체크
    if 폭들[0] > 폭들[1] > 폭들[2] > 폭들[3]:
        # 마지막 구간 거래량이 평균보다 낮은지
        if mean(거래량_시계열[-15:]) < mean(거래량_시계열[-60:]) * 0.7:
            return True
    return False
```
