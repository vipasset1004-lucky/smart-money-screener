# 국내 실전가들 — 한국 시장 특화 수급 매매

## 핵심 주장 (한국 시장의 특수성)

> **"한국 시장은 외국인·기관·개인의 3자 수급 게임이다."**
>
> 미국 대가들의 이론은 **장기 추세·기관 후원** 중심이지만,
> 한국은 외국인 영향이 절대적이고, 단기 수급이 시세를 만든다.

## 한국 시장의 특징

| 항목 | 한국 | 미국 |
|------|------|------|
| **외국인 비중** | 시총의 30~35% | 작음 |
| **개인 비중** | 거래대금의 60~70% | 30% 미만 |
| **단타 비중** | 매우 높음 | 낮음 |
| **상한가 제도** | ±30% | 없음 |
| **공매도** | 일부 종목만 | 자유 |
| **테마 파급력** | 매우 큼 | 중간 |
| **종목 회전율** | 높음 | 낮음 |

→ **외국인·기관 수급 + 테마·재료 + 차트** 3박자가 핵심

## 실전 수급 분석 원칙 (국내 통용)

### 1. 누가 사는지가 가장 중요

**우량 매수 주체 (좋은 수급)**
- 외국인 (장기 추세, 환율 안정 시 강함)
- 연기금 (매우 장기, 패시브 위주)
- 투신 (펀드 환매 영향, 중기)

**경계 매수 주체**
- 사모 (단기 추종, 변덕)
- 기타법인 (정체 불명, 자전 가능성)

**역지표**
- 개인 (대다수 손실)
- 단, "스마트 개미"는 예외

### 2. 외국인 누적 매수의 진정성

**진짜 매수 vs 가짜 매수**

| 구분 | 패턴 | 신뢰도 |
|------|------|--------|
| 외국인 + 기관 동시 매수 | 가장 신뢰 | ⭐⭐⭐ |
| 외국인 단독 + 환율 안정 | 신뢰 | ⭐⭐ |
| 외국인 단독 + 환율 급변 | 헷지 가능성 | ⭐ |
| 종가 동시호가 매수 | 작전 가능성 | ⚠️ |
| MSCI/FTSE 리밸런싱 | 패시브, 의미 적음 | ⚠️ |
| 프로그램 차익 | 의미 없음 | X |

→ 우리 알고리즘: **프로그램 차익 제외 + 동시매수 가중치**

### 3. "한 번 들어온 외국인은 쉽게 나가지 않는다"

- 외국인은 **포지션 진입에 시간을 들이고**, 나갈 때도 천천히 나감
- 따라서 외국인 매수 전환 시점 → 향후 수개월 추세 시작 가능성

### 4. 기관과 외국인의 시간차

- 외국인이 먼저 들어오고, 기관이 따라옴 (코스피 대형주)
- 기관이 먼저 들어오고, 외국인이 늦게 옴 (코스닥 중소형주)
- → **둘 다 들어온 종목**이 가장 강함 (동시매수)

## 한국 시장 텐버거 패턴

### 공통 특성
1. **외국인 1~2년 누적 순매수 양호**
2. **테마성 (성장 산업, 신기술, 정책 수혜)**
3. **실적 동반 성장** (영업이익 급증)
4. **시총 확장 가능성** (소형 → 중형 → 대형)
5. **신고가 돌파 후 추가 가속**

### 사례 (방향성 — 별도 케이스 스터디 필요)
- **에코프로 (2022~2023)**: 2차전지 양극재, 외국인 매집 → 수십 배
- **한미반도체 (2023)**: HBM 본더 독점, 외국인·기관 동시 매수
- **삼성SDI (2020~2021)**: EV 배터리, 장기 매집 후 폭발
- **셀트리온 (2017~2018)**: 바이오 시밀러, 외국인 매수
- **삼성전자 (2020 전후)**: 메모리 반도체 슈퍼사이클

→ 모두 **장기 매집 + 출발 + 강한 펀더멘털**

## 한국 시장 단타 패턴

### 7일 내 폭발의 트리거
1. **외국인·기관 신규 매수 전환**
2. **거래대금 급증 (평균 대비 3배+)**
3. **재료/테마 발생** (정책, 신제품, 실적 가이던스)
4. **신고가 돌파**
5. **수급 공백 없는 강한 양봉**

### 단타 함정 (회피)
- **상한가 후 다음날 갭하락** — 추격 매수 위험
- **고가 음봉 + 거래량 폭발** — 분산 의심
- **공매도 잔고 급증** — 숏 공격 가능
- **펀더멘털 무관 단순 테마주** — 변동성 폭락

→ 우리 알고리즘: **종가 강세 + 펀더멘털 필터** 결합

## 박병창 (실전 수급 매매 대가)

### 핵심 메시지
- "수급은 거짓말 안 한다"
- 5일/20일/60일 외국인·기관 누적 순매수 추적
- 외국인 매수 + 기관 매수 + 개인 매도 → 황금 패턴
- **단순 절대금액보다 시총 대비 비율**이 중요

## 김정환 (외국인·기관 흐름 분석)

### 핵심 메시지
- 외국인·기관 매수 패턴의 시간차 분석
- 종목별 수급 주체별 행동 패턴 추적
- 매도 → 매수 전환 시점 = 변곡점

## 슈퍼개미 일반론

- 자기 매매 일지 작성, 데이터 기반 의사결정
- 손절 룰 엄수 (-5~7%)
- 분할 매수·매도
- **시장 강세 시에만 적극 매매**

## 우리 알고리즘 적용

| 한국 실전 개념 | 우리 코드 |
|----------------|-----------|
| 외국인 + 기관 동시매수 | `수급_일치도` 30점 |
| 시총 대비 비율 | `수급_강도` 25점 |
| 매도 → 매수 전환 | `지속성` 보너스 |
| 5/20/60일 누적 | `수급_점수` 다중 윈도우 |
| 프로그램 차익 제외 | (확장) 차익거래 분리 |
| 외국인 단독 + 환율 체크 | (확장) 환율 필터 |
| 종가 강세 마감 | `단타_출발`의 종가 ≥ (시가+고가)/2 |
| 펀더멘털 필터 | (확장) PER/EPS 체크 |

## 한국 특화 우리 알고리즘 보강 사항

1. **코스닥 중소형주는 프로그램 매매 비중 작음** → 가중치 다르게
2. **연말·결산 시즌엔 기관 윈도우드레싱** 주의 (12월 매수 → 1월 매도)
3. **MSCI/FTSE 정기변경 일정** 반영 (의미 없는 매수 제외)
4. **공매도 재개 종목** 모니터링 (변동성 ↑)

## 추천 추가 학습

- 박병창 강의/저서 — 실전 수급 매매
- 김정환 유튜브 — 외국인·기관 흐름 분석
- 한국거래소 KIND 공시 (정정공시·블록딜)
- 키움증권 영웅문 수급 분석 도구
