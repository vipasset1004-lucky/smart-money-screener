# Stan Weinstein — Stage Analysis

## 핵심 주장

> **"모든 주식은 4단계를 거친다. Stage 2만 사라."**
>
> *Secrets for Profiting in Bull and Bear Markets* (1988).
> Wyckoff의 4단계 사이클을 30주(150일) 이동평균선과 결합해 정량화.

## 4단계 정의 (30주 이평선 기준)

| 단계 | 가격 vs 30주선 | 30주선 기울기 | 거래량 | 행동 |
|------|---------------|---------------|--------|------|
| **Stage 1: Basing** | 30주선 근처 횡보 | 평탄 | 낮음 | 관찰 |
| **Stage 2: Advance** | 가격 > 30주선, 30주선 상승 | **상승** | 증가 | **매수** ⭐ |
| **Stage 3: Top** | 30주선 근처 횡보 | 평탄 | 변동 | 매도 준비 |
| **Stage 4: Decline** | 가격 < 30주선, 30주선 하락 | **하락** | 변동 | 절대 매수 X |

## Stage 1 → Stage 2 전환 (텐버거 출발 신호의 핵심)

### Weinstein의 정의
```
Stage 2 진입 조건:
✅ 가격이 30주(150일) 이평선 위로 돌파
✅ 30주 이평선이 평탄 → 상승으로 전환
✅ 돌파일 거래량 평균 대비 2배+
✅ 매집 박스권 상단을 명확히 이탈 (1~2% 이상)
✅ Mansfield RS Rating 양수 전환
```

### 우리 알고리즘과 매칭

| Weinstein | 우리 코드 |
|-----------|-----------|
| 30주 이평선 위 돌파 | `60일선 > 240일선` (한국식 단축) |
| 30주선 상승 전환 | (보강) `MA(150).slope > 0` 체크 |
| 거래량 2배+ | `거래량 폭발` 조건과 일치 |
| 매집 박스 이탈 | `박스권_상단_돌파` |
| Mansfield RS 양수 | RS 상위권 (신고가 스크리너의 RS상위20%와 동일) |

## Mansfield Relative Strength

> **"종목의 코스피 대비 상대 강도. 양수 = 시장보다 강함."**

```
Mansfield RS = (종목수익률 / 코스피수익률) - 1
            (52주 기준)
```

음수 → 양수 전환되는 시점이 텐버거 출발의 좋은 동행 신호.

## 매도 신호 (Stage 3 → Stage 4)

| 신호 | 의미 |
|------|------|
| 가격이 30주선 아래로 이탈 | 즉시 매도 검토 |
| 30주선이 평탄 → 하락 전환 | 추세 종료 |
| 신저가 갱신 | Stage 4 확정 |
| 거래량 동반 하락 | 분산 진행 |

## "Stage 2 외엔 절대 사지 마라"

Weinstein의 핵심 메시지:
- **Stage 1 매수 = 너무 일찍** → 시간 낭비
- **Stage 2 매수 = 정답** → 추세 추종
- **Stage 3 매수 = 천장 잡기** → 위험
- **Stage 4 매수 = 떨어지는 칼날** → 절대 금지

→ 우리가 "출발 게이트"를 둔 이유와 정확히 일치

## 우리 알고리즘 적용 (직접)

**텐버거 출발 = Stage 1 → Stage 2 전환**
```python
def is_stage1_to_stage2(종목):
    return (
        종목.가격 > 종목.MA150  # 가격이 30주선 위
        and slope(종목.MA150, 30) > 0  # 30주선 상승 중
        and 종목.오늘_거래량 >= 종목.평균_거래량 * 2.0  # 거래량 폭발
        and 종목.박스권_상단_돌파  # 매집 박스 이탈
        and 종목.RS상대강도 > 0  # 시장 대비 강함
    )
```

**Stage 4 회피**
- 텐버거 후보라도 가격이 MA150 아래면 후보에서 제외

## Stage 1 (매집) 식별

```python
def is_stage1(종목, lookback=150):
    가격 = 종목.종가시리즈[-lookback:]
    범위 = (max(가격) - min(가격)) / mean(가격)
    
    return (
        범위 < 0.20  # 20% 이내 박스
        and abs(slope(종목.MA150, 30)) < 0.0001  # MA 평탄
    )
```

→ 우리 "매집중" 탭의 정의로 사용 가능

## 한국 시장 적용 시 주의

- 30주선 = 150일선이 한국 시장에선 **너무 김** (변동성 더 큰 시장)
- 한국 적용: **20주선(100일) 또는 60일선** 활용 권장
- 우리는 보수적으로 60일선 + 240일선 조합 (신고가 스크리너 호환)
- Mansfield RS는 코스피 대비 (단, 코스닥 종목은 코스닥 지수 대비도 함께 고려)
