# Jesse Livermore — Pivotal Point & 추세 추종

## 핵심 주장

> **"피벗 포인트(Pivotal Point)에서 진입하라. 그 외엔 손실이다."**
>
> *Reminiscences of a Stock Operator*, *How to Trade in Stocks* (1940).
> 20세기 최고의 트레이더. 단타와 장기 매매를 자유자재로 구사한 원조.

## Pivotal Point의 정의

> **"오랜 횡보·매집 후, 명확한 신고가 돌파 또는 추세 전환이 일어나는 결정적 가격."**

### 두 종류

**1. Continuation Pivotal Point (지속형 피벗)**
- 상승 추세 중간, 박스권 횡보 후 돌파
- → **단타 출발**과 매칭

**2. Reversal Pivotal Point (전환형 피벗)**
- 장기 횡보 또는 하락 추세 후 추세 전환
- 신고가 돌파 또는 신저가 회복
- → **텐버거 출발 (Stage 1→2 전환)**과 매칭

## Livermore의 진입 룰

```
1. 종목이 Pivotal Point에 도달할 때까지 인내한다
2. 도달 후 거래량이 폭발하면 즉시 진입
3. 만약 즉시 반대로 움직이면 (당일/다음날) 즉시 손절
4. 진입 후 추세 따라 피라미딩 (분할 추가 매수)
```

### 피라미딩 (Pyramiding)
- 첫 진입 후 가격이 +10~20% 상승하면 추가 매수
- 단, **추가 매수가는 항상 이전보다 높아야 함**
- 평균 매수가가 시장가 아래에서 유지됨 → 손절선 자동 상향

## "추세는 친구다, 끝까지 가라"

> **"내 큰돈은 매매가 아니라 들고 있는 데서 나왔다."**

- 단타로 들어가도, 추세가 좋으면 **장기 보유로 전환**
- 우리 알고리즘의 "황금자리(단타+텐버거 동시)" 개념과 일치

## "절대 평균 매수단가를 낮추지 마라"

> **"떨어지는 종목을 사는 건 자살 행위다."**
- 손실 종목 추가 매수 금지
- 수익 종목에만 추가 매수 (피라미딩)

→ 우리 알고리즘은 **하락 종목 자체를 추천에서 배제** (Stage 4 회피)

## "Lone Wolf" 원칙

- 시장의 합의(컨센서스)를 따르지 마라
- 자기 분석 결과만 믿어라
- 군중 심리 = 손실 원인

## Livermore의 시장 분석 방법

**1. 시장 전반 (Tape Reading)**
- 전체 시장 추세 먼저 확인
- 강세장에선 매수, 약세장에선 매도
→ 우리 (확장) 시장 레짐 필터

**2. 업종 그룹 (Group Movement)**
- 한 종목이 아닌 같은 업종 여러 종목이 함께 움직이는지 확인
- 업종 1등주가 가장 강한 종목
→ 텐버거 후보 선별 시 업종 동조성 체크 (확장)

**3. 개별 종목**
- Pivotal Point 대기
- 거래량 확인
- 진입

## 우리 알고리즘 적용

| Livermore 개념 | 우리 코드 |
|----------------|-----------|
| **Continuation Pivot** | `단타_출발` 신호 (박스권 돌파 + 거래량) |
| **Reversal Pivot** | `텐버거_출발` 신호 (Stage 1→2 + 신고가) |
| **거래량 동반 필수** | 모든 출발 신호의 공통 조건 |
| **즉시 손절** | (확장) -7~8% 자동 손절선 표시 |
| 피라미딩 | (확장) 단계별 추가매수 가이드 |
| 추세 추종 | "황금자리" 라벨 (단타→텐버거 전환 가능) |

## 단타 출발 = Continuation Pivot

```python
def is_continuation_pivot(종목):
    """단타용: 단기 박스권 돌파 + 거래량"""
    return (
        종목.가격 > 종목.단기박스_상단  # 박스 돌파
        and 종목.당일_거래대금 >= 종목.평균_거래대금 * 3.0
        and 종목.종가 >= (종목.시가 + 종목.고가) / 2  # 강한 마감
    )
```

## "당일 안 움직이면 손절" 룰

Livermore의 핵심: **출발 신호 후 즉시 추세가 나와야 진짜**

→ 우리 알고리즘 확장 아이디어:
- 단타 진입일 다음날 종가가 진입가 아래면 손절
- 즉, **24~48시간 내 검증 룰**
